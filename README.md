# Banking Risk Analytics Dashboard

An end-to-end Banking Risk Analytics project that combines Python for Exploratory Data Analysis (EDA) and Power BI for interactive dashboarding. The project analyzes customer banking behavior, loans, deposits, and financial metrics to provide actionable insights for banking institutions.

## Project Objectives
- Analyze customer banking behavior
- Identify trends in loans and deposits
- Perform data cleaning and feature engineering
- Build interactive Power BI dashboards
- Support data-driven lending and business decisions

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Power BI
- DAX

## Key Features
- Data cleaning and preprocessing
- Feature engineering (Income Band, Engagement Days, Processing Fees)
- Exploratory Data Analysis (EDA)
- Interactive dashboards
- Loan Analysis
- Deposit Analysis
- Customer Summary Dashboard
- KPI tracking for Loans, Deposits, Clients, Fees, and Credit Cards

## KPIs
- Total Clients
- Total Loans
- Total Deposits
- Bank Loans
- Business Lending
- Total Fees
- Bank Deposits
- Savings Account Amount
- Checking Account Amount
- Foreign Currency Amount
- Credit Card Amount
