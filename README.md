# Banking Risk Analytics Dashboard

An end-to-end Banking Risk Analytics project that combines **Python** for Exploratory Data Analysis (EDA) and **Power BI** for interactive dashboard development. The project analyzes customer banking data to generate insights into loans, deposits, customer engagement, and banking performance.

## Project Overview

Banks generate massive amounts of customer and financial data every day. This project demonstrates how data analytics can be used to transform raw banking data into meaningful business insights that support better decision-making and risk assessment.

The project consists of:

- Data Cleaning & Preprocessing using Python
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Interactive Power BI Dashboard
- KPI and Business Metrics Visualization

## Data Cleaning & Feature Engineering

The following transformations were performed:

- Created **Engagement Days**
- Created **Engagement Timeframe**
- Created **Income Band** (Low / Mid / High)
- Calculated **Processing Fees**
- Removed inconsistencies and prepared data for visualization
